# Robosys2021_Task1

---

2021年度ロボットシステム学7～8回にて作成したコードを改変し、

7セグメントLEDで1～9までの値をほぼランダムに表示するようにしたデバイスドライバです。

選択肢が多くて困ったときに役立つと思います。

---

# 確認済み動作環境

- Raspberry Pi4 Model B 8GB OKdo版

- OS : Ubuntu 18.04 LTS

---

# 用意する物

- 1kΩ抵抗 × 7

- ジャンパー線（オス-メス）× 8

- ジャンパー線（オス-オス）× 13

- 7セグメントLED × 1

- ブレッドボード × 1

- Raspberry Pi4 Model B × 1
